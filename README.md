
# HireReady 🚀

AI-powered resume builder and job matching platform built with React, Node.js, Firebase and Google Generative AI.

## Features ✨
- **User Authentication:** Secure signup and login using Firebase
- **Resume Upload & Parsing:** Upload resumes in PDF or DOCX format
- **AI Resume Analysis:** AI-driven feedback and scoring on resumes
- **AI Resume Builder:** Generate professional resumes using AI
- **Job Matching:** Match resumes against job descriptions
- **Cover Letter Generation:** Auto-generate cover letters

## Technologies Used 🛠️
**Frontend:** React, TypeScript, Vite, Tailwind CSS, shadcn/ui

**Backend:** Node.js, Express.js, TypeScript, Firebase, Google Generative AI

## Installation
1. Clone the repository
2. Run `npm install` in root folder
3. Run `npm install` in backend folder
4. Add `.env` files with your Firebase and Google AI keys
5. Run `npm run dev` in both folders
